Class SlideTabUI

UI class which controls the sliding function of the home screen. Also all UI elements within those pages, which means almost everything in the app.

Uses a modified version of the slider library lory (http://meandmax.github.io/lory/) to control sliding between pages, and the UIUtil.UIItem system to dynamically build the pages at runtime.

Note: onFocus of children elements is called when the slide finishes transitioning.
